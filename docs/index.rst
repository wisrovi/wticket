WTicket Documentation
=====================

Welcome to the **WTicket** documentation!

WTicket is an enterprise-grade, serverless ticket management system engineered with vanilla JavaScript and JSONBin.io. Designed for zero-infrastructure deployment on static hosting platforms, it delivers production-ready functionality without the complexity of traditional backend systems.

.. image:: _static/logo.svg
   :width: 200
   :align: center
   :alt: WTicket Logo

Features
--------

* **Zero Infrastructure**: Deploy on any static hosting provider
* **Serverless Architecture**: No backend servers required
* **Real-time Statistics**: Live dashboard with ticket metrics
* **Role-Based Access**: Administrator and user panels
* **Progressive Web App**: Installable on any device
* **Offline Support**: Service Worker caching
* **Multi-language**: Spanish and English support
* **Dark Mode**: Built-in theme customization

Quick Start
-----------

Get up and running in under 5 minutes:

.. code-block:: bash

   # Clone the repository
   git clone https://github.com/wisrovi/wticket.git
   cd wticket

   # Start a local server
   python3 -m http.server 8000

   # Open in browser
   # http://localhost:8000

For detailed installation instructions, see :doc:`getting_started/index`.
